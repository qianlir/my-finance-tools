### 2️⃣ 中国移动(600941) — **高息防御型**

**现状**: 93.52 元，典型的高股息蓝筹  
**周期**: 稳定高现金流，业绩确定性强  
**操作**: 长期持有，作为仓位锚

| 指标 | 评价 |
|------|------|
| 估值 | PE 中等，看股息率更重要 (5.5%) |
| 技术 | 低波动率，防御性强 |
| 基本面 | 5G/AI数据中心增长，确定性业绩 |
| 分红 | 连续高派息，机构标配 |
| 风险 | 政策监管 (电信行业特性) |

**五一前策略**:
- ✅ **不动** (长期持有)
- 📍 目标: 占总仓位 30-40%，作为底仓稳定器
- 💰 享受高股息 (年化 5.5%)

**为什么是"锚"?**
- 高股息可抵消短期波动
- 业绩稳定性强于五粮液
- 低风险特性适合压舱石

### 3️⃣ 阿里巴巴(09988.HK) — **低估反转型**

**现状**: 港股通持仓，数据采集失败，基于基本面判断  
**周期**: 中概股底部区域  
**操作**: 波段操作，高抛低吸

| 指标 | 评价 |
|------|------|
| 估值 | PE ~10-12x，远低于美股阿里 (BABA) |
| 基本面 | 电商基本盘被分流，云业务增长亮点 |
| 分红 | 积极回购+分红，股东回报意愿强 |
| 风险 | 中概股地缘政治风险，政策不确定 |
| 机会 | 低估值+大规模回购，底部反弹潜力 |

**五一前策略**:
- 🔄 **波段操作** — 不长期死持
- 📊 目标持仓占比: 20-30%
- 🎯 操作思路: 
  - 港股通南向资金充足 → 可持有
  - 若出现明显利空 → 止损
  - 若反弹到合理估值 → 卖出部分获利

**为什么是"波段"?**
- 中概股政策风险大 → 不适合长期死持
- 低估值反弹可期 → 短期有机会
- 汇率波动影响 → 需要主动管理
