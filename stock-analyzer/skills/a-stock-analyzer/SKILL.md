# a-stock-analyzer — A股/港股通个股深度分析

## 能力
- 采集实时行情（价格、涨跌幅、成交额）
- 获取近5年财务数据（营收、净利、毛利率、ROE）
- 获取分红记录，计算股息率
- 计算技术指标（MA5/20/60/120/250、MACD、RSI）
- 生成综合分析报告（估值判断 + 技术面判断 + 操作建议）

## 使用流程

### 1. 采集数据
```bash
python scripts/collect_data.py --code <股票代码> --market <A股|港股>
```

### 2. 生成分析报告
基于采集到的数据，由 Agent 进行综合分析并输出报告。

## 报告模板

分析报告包含以下模块：

1. **估值快照**: PE/PB/股息率 + 历史百分位
2. **基本面分析**: 营收/净利增速趋势、毛利率变化、ROE趋势
3. **技术面分析**: 均线系统状态、趋势判断、支撑压力位
4. **综合判断**: 估值水位 + 技术面趋势 → 操作建议
5. **风险提示**: 主要风险因素
