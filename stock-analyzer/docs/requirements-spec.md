# 需求规格书 — A股分析 Agent

## 客户信息
- 用户: 个人投资者
- 目标: 对持仓A股/港股进行结构化分析，支持估值判断、技术面分析和操作建议
- 频率: 需要时触发，约每周1-2次

## 核心目标 (Goals)

### G1: 一键获取个股全景数据
采集实时行情、财务指标、分红数据、技术均线，输出结构化快照。

### G2: 基本面估值分析
基于年报财务数据，计算PE/PB/PEG/股息率，与历史区间对比，判断估值水位。

### G3: 技术面趋势判断
计算均线系统(MA5/20/60/120/250)、MACD、RSI，判断当前趋势和支撑压力位。

### G4: 综合分析报告
输出结构化Markdown报告，包含估值判断、技术面判断、风险提示和操作建议。

## 需求分解

| # | 需求 | 类型 | 优先级 |
|---|------|------|--------|
| R1 | 数据采集脚本(行情+财务+分红+均线) | Script | P0 |
| R2 | 估值分析引擎(PE/PB/股息率+历史百分位) | Script | P0 |
| R3 | 技术面分析(均线+趋势+支撑压力) | Script | P0 |
| R4 | 综合分析报告生成 | Agent | P0 |
| R5 | 港股通数据支持 | Script | P1 |

## 约束
- 数据源: akshare（免费，A股/港股均支持）
- 依赖: Python 3.9+, akshare, pandas, numpy
- 输出: Markdown报告 + JSON数据
