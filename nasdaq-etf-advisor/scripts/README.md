# ETF数据采集和推荐脚本

## 核心脚本（2个）

### 1. update_data.py - 数据更新
```bash
python3 update_data.py              # 更新所有数据（历史+实时）
python3 update_data.py --realtime   # 只更新实时数据
python3 update_data.py --history    # 只更新历史数据
```

功能：
- 获取ETF历史价格（腾讯/新浪API）
- 获取ETF历史净值（东方财富API）
- 获取实时价格和净值
- 保存到数据库

### 2. recommend_by_change.py - 基于涨幅偏离的ETF推荐
```bash
python3 recommend_by_change.py
```

功能：
- 计算每个ETF的显示溢价、历史平均溢价、当天涨幅
- 计算平均涨幅（作为纳指/标普的基准涨幅）
- 计算超额溢价 = 显示溢价 + 当天涨幅 - 平均涨幅 - 历史平均溢价
- 按超额溢价排序，给出买入/卖出建议

## 推荐逻辑

```
平均涨幅 = Σ(各ETF涨幅) / n
超额溢价 = 显示溢价 + 当天涨幅 - 平均涨幅 - 历史平均溢价
```

- 超额溢价 < 0 → 买入（✅数量根据偏离程度）
- 超额溢价 > 0.3% → 卖出（❌）
